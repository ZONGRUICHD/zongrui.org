# zongrui.org

ZongRui 的个人介绍网站。以深色技术编辑风呈现 Rust、机器人、嵌入式、Linux、网络基础设施、AI 工具与网页作品。

## 本地开发

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

## Cloudflare Pages

- Production branch: `main`
- Build command: `npm run build`
- Build output directory: `dist`
- Root directory: `/`

## 内容说明

网站文案来自公开项目资料与用户明确授权的 ChatGPT 账号记忆摘要，并经过隐私筛查。仓库不保存 ChatGPT 对话、联系方式、学校信息或其他私密资料。
